## CPU采样策略

1. 利用 sleep + warmup 机制避免初始加载时数据失真。
2. 统计 P/E core 数量，显示在 “核心使用率” 的Lable 旁边，如：“核心使用率（4P+6E）”
3. 核心使用率的进度条列表，分组排序性能核心和能效核心。先显示性能核心，然后再显示能效核心。
4. 增加 Load Average，显示在概览页面，放在温度上面。如：
    负载: 3.2 / 2.8 / 2.5
    温度:   37 °
